# 计算稿纸 - uTools 插件

一个简洁的 uTools 计算器插件，支持数学计算并自动保存历史记录。

## 功能特性

- **快速计算**：输入数学表达式，按回车立即得到结果
- **历史记录**：自动保存最近 20 条计算记录
- **单条删除**：鼠标悬停可删除单条记录
- **清空功能**：一键清空所有历史记录
- **稿纸风格**：仿稿纸网格背景，简洁美观

## 支持的运算

- 基础四则运算：`+` `-` `*` `/`
- 乘方运算：`^`
- 括号：`()`
- 取模：`%`
- 小数运算

## 使用示例

```
3 + 5 = 8
(10 + 20) * 2 = 60
100 / 3 = 33.3333333333
2^10 = 1024
```

## 安装方法

1. 下载并安装 [uTools](https://u.tools/)
2. 下载并安装 [uTools 开发者工具](https://u.tools/)
3. 克隆本仓库到本地
4. 打开 uTools 开发者工具，选择 `plugin.json` 文件加载插件
5. 在 uTools 中输入"计算稿纸"或"calc"即可使用

## 开发说明

本插件采用原生 HTML/CSS/JavaScript 开发，无需任何构建工具，轻量简洁。

在开发环境中使用 `localStorage` 存储数据，发布到 uTools 后自动切换为 `utools.db` 存储。
